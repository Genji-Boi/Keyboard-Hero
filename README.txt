**Download zip file and open the file titled Keyboard Clicker Version 1.0.exe to start playing**
Made using Unity Game Engine
Written in C#
Clicker game based on concept of mechanical keyboard switches.
Upgrade, plan, and click to garner as many keyboard clicks as possible.
